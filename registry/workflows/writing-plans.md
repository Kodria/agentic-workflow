# Writing Plans

> [!IMPORTANT]
> Este workflow genera un plan de implementación detallado. Sigue estrictamente la skill `writing-plans`.

Este workflow genera un plan de trabajo detallado a partir de una idea de desarrollo ya identificada y refinada.

## Pasos

1. Leer las instrucciones de la skill `writing-plans` ubicada en la carpeta de skills (`~/.agents/skills/writing-plans/SKILL.md`). Usa la herramienta `view_file` para leer el archivo `SKILL.md` de la skill.
2. Seguir **todos** los pasos definidos en la skill `writing-plans` sin omitir ninguno.
3. Generar el plan de trabajo completo según las instrucciones de la skill.
4. Guardar el plan en el formato y ubicación que indique la skill.
5. La skill ofrecerá al usuario opciones de ejecución (Subagent-Driven o Parallel Session). Seguir la decisión del usuario.

## Notas

- Usar ÚNICAMENTE la skill `writing-plans` para generar el plan.
- El plan generado por esta skill será la base para la ejecución posterior.
- Obedecer estrictamente los pasos, formato y estructura definidos por la skill.
