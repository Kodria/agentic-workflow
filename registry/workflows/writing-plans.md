# Writing Plans

> [!IMPORTANT]
> **Modo de Agente**: Use **Fast Mode**. No genere planes de implementación internos de Antigravity (`implementation_plan.md`); salte directamente a la ejecución de la skill `writing-plans`.

Este workflow genera un plan de trabajo detallado a partir de una idea de desarrollo ya identificada y refinada.

## Pasos

1. Leer las instrucciones de la skill `writing-plans` ubicada en la carpeta de skills globales (`~/.agents/skills/writing-plans/SKILL.md`). Usa la herramienta `view_file` para leer el archivo `SKILL.md` de la skill.
2. Seguir **todos** los pasos definidos en la skill `writing-plans` sin omitir ninguno.
3. Generar el plan de trabajo completo según las instrucciones de la skill.
4. Guardar el plan en el formato y ubicación que indique la skill.

## Restricciones

- **OBLIGATORIO**: Usar ÚNICAMENTE la skill `writing-plans` para generar el plan. No usar ninguna otra skill, herramienta de planificación propia de Antigravity, ni proceso alternativo.
- El plan generado por esta skill tiene prioridad sobre cualquier planificación interna del agente.
- Obedecer estrictamente los pasos, formato y estructura definidos por la skill, ya que el plan resultante será la base para la ejecución posterior.
