# Documenting Modules

> [!IMPORTANT]
> **Modo de Agente**: Use **Fast Mode**. El objetivo es ejecutar la documentación de manera eficiente después del desarrollo.

Este workflow automatiza la creación de documentación del sistema analizando los planes y el código desarrollado.

## Pasos

1.  Leer las instrucciones de la skill `documenting-modules` ubicada en la carpeta de skills globales (`~/.agents/skills/documenting-modules/SKILL.md`). Usa la herramienta `view_file`.
2.  Seguir **todos** los pasos definidos en la skill `documenting-modules`.
3.  Generar los archivos de documentación en `docs/modules` y actualizar el `README.md`.

## Notas

-   Ejecutar este workflow inmediatamente después de terminar un desarrollo (p.ej. después de `executing-plans`).
-   Asegurar que la documentación esté en español.
