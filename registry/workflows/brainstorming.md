# Brainstorming

> [!IMPORTANT]
> Este flujo requiere investigación profunda, colaboración interactiva y la creación de documentos de diseño.

Este workflow inicia y guía el proceso completo de brainstorming para planificar un nuevo desarrollo.

## Pasos

1. Leer las instrucciones de la skill `brainstorming` ubicada en la carpeta de skills (`~/.agents/skills/brainstorming/SKILL.md`). Usa la herramienta `view_file` para leer el archivo `SKILL.md` de la skill.
2. Seguir **todos** los pasos definidos en la skill `brainstorming` sin omitir ninguno. No saltar, resumir ni modificar el proceso descrito en la skill.
3. Continuar la ejecución interactiva con el usuario hasta completar todos los pasos del flujo de brainstorming.
4. Al finalizar, asegurarse de que los resultados queden guardados en un documento `.md` según lo indique la skill.
5. La skill de brainstorming encadenará automáticamente hacia `writing-plans`. Permitir que el encadenamiento natural ocurra.

## Notas

- Este flujo es **interactivo**: se requiere la participación del usuario en cada paso.
- No uses ninguna otra skill ni proceso de planificación alternativo. Solo sigue lo definido en `brainstorming`.
- Si la skill requiere generar preguntas, análisis o entregables intermedios, ejecútalos antes de avanzar al siguiente paso.
