# Docs System Orchestrator Workflow

1. Lee detenidamente el manifiesto de la skill orquestadora ubicado en `~/.agents/skills/docs-system-orchestrator/SKILL.md` usando la herramienta `view_file`.
2. Sigue las **Instrucciones de Ejecución** de la skill al pie de la letra.
3. Interactúa con el usuario para sugerirle el próximo comando a ejecutar en base a sus necesidades y el modelo de negocio.
