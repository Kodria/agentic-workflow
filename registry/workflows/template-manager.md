# Gestión de Plantillas de Documentación

1. Usa tus herramientas de búsqueda de archivos para localizar dinámicamente la skill `template-manager` (buscar `template-manager/SKILL.md` en las carpetas de skills disponibles). Leer el archivo `SKILL.md` encontrado con `view_file`.
2. Sigue estricta y secuencialmente los pasos allí descritos. Si el usuario ingresó un requerimiento inicial, aplícalo de inmediato iniciando con la lectura de las plantillas existentes para evaluar similitudes.
