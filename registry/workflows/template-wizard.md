# Asistente de Plantillas de Documentación

1. Usa tus herramientas de búsqueda de archivos para localizar dinámicamente la skill `template-wizard` (buscar `template-wizard/SKILL.md` en las carpetas de skills disponibles). Leer el archivo `SKILL.md` encontrado con `view_file`.
2. Sigue los pasos descritos secuencialmente. El Paso 0 (Autodescubrimiento) se encargará de ubicar la carpeta de plantillas automáticamente.
